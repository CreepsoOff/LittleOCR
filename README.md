# 🧠 Windows OCR Viewer (Apple Live Text-like)

A clean, native Windows desktop application that provides OCR (Optical Character Recognition) capabilities similar to Apple's Live Text — but fully offline.

## ✨ Features

* 🖼️ Open images (PNG, JPG, JPEG, BMP)
* 🔍 Detect text directly from images
* 🧠 Fully offline OCR (no cloud)
* 🧩 Overlay detected text on the image
* ✂️ Copy all or selected text
* 🎯 Simple and clean UI

## 🎯 Goal

This project aims to recreate a smooth "Live Text" experience on Windows:

* Open an image
* Click an OCR button
* Interact with detected text directly

## 🛠️ Tech Stack

* C#
* .NET 8
* WinUI 3 (or WPF if needed)
* Windows OCR API (preferred)
* Tesseract (fallback)

## 🚀 Getting Started

### Prerequisites

* Windows 10/11
* Visual Studio 2022 or newer
* .NET 8 SDK

### Run the project

```bash
git clone https://github.com/CreepsoOff/LittleOCR.git
cd LittleOCR
```

Then open the solution in Visual Studio and run the project.
